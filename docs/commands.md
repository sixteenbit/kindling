# Commands

## Default task to build the project

```bash
gulp
```

## Builds the project

```bash
gulp build
```

## Runs the build task and watches for file changes

```bash
gulp watch
```

## Runs the styles task

```bash
gulp styles
```

## Runs the javascript task

```bash
gulp javascript
```

## Starts Browsersync

```bash
gulp server
```

## Copy theme to dist folder for production

```bash
gulp release --production
```
